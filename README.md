# Rust Todo Web App

## 项目简介

本项目是一个基于 Rust 开发的 Todo Web 应用，采用前后端分离架构，实现任务的添加、查询与状态管理等功能。

项目主要用于学习 Rust Web 后端开发、REST API 设计以及现代 Web 项目的开发流程。

---

## 技术栈

### 后端

* Rust
* axum
* tokio
* serde

### 前端

* HTML
* CSS
* JavaScript

### 数据库

* SQLite（开发中）

---

## 当前功能

* 获取 Todo 列表
* 添加 Todo
* 前后端通信
* Git 版本管理

---

## 项目结构

frontend 负责页面展示与交互。

backend 负责 REST API 与业务逻辑。

SQLite 用于数据持久化。
